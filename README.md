# Portfólio Pessoal

##  Aluno: Carlos Henrique Michalak

##  Descrição
Portfólio pessoal desenvolvido como projeto prático para a UC Publicar Aplicações Web. Site responsivo contendo informações profissionais, habilidades técnicas e projetos.

##  Principais Pontos Revisados

###  HTML Semântico
- Estrutura semântica com header, nav, section, footer
- Formulário de contato acessível
- Meta tags para responsividade

###  CSS Moderno
- Grid Layout e Flexbox
- Design responsivo com media queries
- Animações e transições CSS
- Variáveis CSS para consistência

###  JavaScript
- Smooth scroll para navegação
- Validação de formulário
- Animações interativas
- Manipulação DOM

### Publicação
- Versionamento com Git
- Deploy no Vercel
- Estrutura organizada de arquivos
